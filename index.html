Início



Objetivo:

Calcular equações básicas presente no nosso cotidiano escolar!

Como usar:

Digite a expressão matemática usando os seguintes sinais:

+ → adição (ex: 2+3)
- → subtração (ex: 5-2)
* → multiplicação (ex: 2*3)
/ → divisão (ex: 6/2)
^ → potência (ex: x^2 ou 2^3)
sqrt() → raiz quadrada (sqrt(9))

Você também pode usar parênteses:
( ) → para organizar contas (ex: 2*(3+4))

Exemplos válidos:
2+2
3*5
2^2+2(2)+sqrt(25)
(1+2)*3
